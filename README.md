# 🤖 NOVA AI - Intelligent Voice Assistant

NOVA AI is an AI-powered voice assistant built with **Python** that enables hands-free interaction through natural language. It leverages **Groq's Llama 3.1**, **SpeechRecognition**, and **pyttsx3** to understand voice commands, provide intelligent responses, and automate everyday tasks.

---

## ✨ Features

- 🎙️ Voice-controlled AI assistant
- 🤖 AI-powered responses using **Groq Llama 3.1**
- 🗣️ Speech-to-Text with SpeechRecognition
- 🔊 Text-to-Speech using pyttsx3
- 🌐 Open websites like Google, YouTube, LinkedIn, WhatsApp, and Instagram
- 🎵 Play music through voice commands
- ⚡ Fast and lightweight Python implementation
- 🔧 Modular and easy-to-extend architecture

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Nova.git
```

### 2. Navigate to the project

```bash
cd Nova
```

### 3. Create a virtual environment

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Create a `.env` file

```env
GROQ_API_KEY=your_groq_api_key
```

### 6. Run the application

```bash
python main.py
```
